# django-example
django example

python V3.7.4

````
git clone https://github.com/wachira90/django-example.git

cd django-example/

virtualenv venv

# LINUX
source env/bin/activate

# WINDOWS
env\Scripts\activate

pip install -r req.txt

cd  django-example/

python manage.py runserver 0.0.0.0:8000
````
